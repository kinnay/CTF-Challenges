* **Name:** Game Hacking
* **Category:** Web
* **Difficulty:** Medium
* **CTF:** CTC 2025

Test your game hacking skills in this challenge! Move through walls, fly through the air, or be creative. Can you obtain the trophy?

Hints:
    * Spawning an instance is not required. The challenge can also be solved locally using the attachment.
    * Reverse engineering the game would be a pain. Try to come up with an easier way to cheat.

*(this challenge was made for CTC 2025, a beginner friendly CTF whose theme was "Security Samurais")*
