
Reverse engineering the game has intentionally been made difficult. The intended solution is to use [GameConqueror](https://github.com/scanmem/scanmem) instead, or a similar tool.

Notes:
* Searching for 'number' makes the tool search for both integers and floats.
* The spikes can be bypassed by placing the frog above the flag, and quickly moving its position again after it touches the flag before it touches the spikes. This requires good timing.
